### 防火墙设置
1. 关闭宿主机的防火墙：systemctl stop firewalld.service,
2. 查看防火墙状态：firewall-cmd --state
3. 禁用防火墙：systemctl disable firewalld.service

### docker容器设置
1. 设置开机启动：systemctl enable docker
